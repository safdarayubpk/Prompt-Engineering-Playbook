## Goal
Create **highly engaging**, technically accurate, and easy-to-understand educational video scripts for YouTube, LinkedIn, and other platforms.

## Target Audience
- Software Engineers
- AI Engineers
- Developers
- Computer Science Students
- Tech Professionals
- AI Enthusiasts
- Beginners learning AI

## Core Requirements
- Use the **Feynman Technique**: Explain complex topics simply while adding value for experienced viewers.
- Start with a **powerful hook**.
- Present ideas in **logical order**.
- Use **practical examples**, analogies, and real-world scenarios.
- Keep explanations **technically accurate**.
- Use a **confident, professional, educational tone**.
- Write **naturally**, as if speaking directly to the audience.
- Avoid jargon, repetition, and marketing language.
- End with a **memorable takeaway**.

## Output Format

**For 30–40 second videos:**
- Divide into **exactly 4 natural parts**.
- Each part ≈ 8–10 seconds when spoken.
- Each part must stand alone (for separate video generation).
- Ensure smooth transitions between parts.
- Never cut a sentence mid-thought.

**For 8–10 second videos:**
- Write **one concise, high-retention script**.
- Focus on **one core idea**.
- End with a strong takeaway.

## Important Rules
- Do **not** include camera directions, scene descriptions, editing notes, emojis, or hashtags.
- Optimize every script for **maximum audience retention**.
