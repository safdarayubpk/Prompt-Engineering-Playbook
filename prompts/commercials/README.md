# Commercials & Advertisement Prompts

This folder contains reusable prompt templates for creating high-converting advertisement videos, product commercials, brand stories, SaaS demos, and promotional content using YouTube AI Playground and other video generation tools.

## Best Practices
- Focus on emotional connection + clear value proposition
- Use strong hooks in the first 3 seconds
- Include compelling calls-to-action
- Maintain brand consistency across videos
- Optimize for both Shorts and long-form formats

## Files in this folder
- `product_launch.md` — Prompts for new product introductions
- `service_promotion.md` — Service and SaaS promotion
- `brand_story.md` — Emotional brand storytelling
- `demo_video.md` — Product demonstration scripts & visuals
